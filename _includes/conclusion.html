Responsive icon design

* One more problem worth mentioning is that finer details may get lost as a design is scaled down * Want to design for size used
* Here we can see the microphone icon from the set Iconic. * They're trying to create general-use icons that look excellent at any size. * What you can see them doing is thinking about the amount of detail you can have in an icon at any given size. * You probably won't have the time to design all of these, or even use all of these * But, it is important to try to understand how your icons will be used. There is a big difference between an icon used at 25, 50, or 100 pixels.
